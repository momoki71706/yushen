# 亲密设备桥接程序（SVAKOM SX014A）

设备是蓝牙连电脑的，云端后端够不到它，所以需要这个小程序在你电脑上当“中间人”：
它一边连设备蓝牙，一边每隔约 0.35 秒问云端“现在该震多强”，再写给设备（并持续
重发保活，否则设备只会震一下就停）。

## 一次性设置

1. 装 Python：到 python.org 下载安装，安装时勾选 **Add Python to PATH**。
2. 打开命令行（PowerShell），安装两个库：
   ```
   pip install bleak requests
   ```
3. 打开 `svakom_bridge.py`，把顶部的 `BACKEND_URL` 和 `TOKEN` 改成你自己的
   （在 App 侧边栏「亲密控制」面板 → 「电脑端桥接设置」里能看到并复制）。

## 使用

```
python svakom_bridge.py
```

- 保持这个窗口开着。使用时：电脑开着、设备开机、设备在电脑蓝牙范围内（约 10 米）。
- **注意**：设备蓝牙同时只能连一处。用这个程序前，先在手机 SVAKOM App 里断开连接。
- 想彻底停止：直接关掉这个窗口，设备立刻不再被控制（最安全的“急停”）。

## 安全设计

- App 里的「强度上限」屿深和手动滑条都突破不了。
- 每条指令都有自动过期时间：万一断网或程序卡住，设备会自动停，不会一直震。
- 桥接程序连不上云端时，一律按“停止”处理（失败即停）。
- App 里关掉「开启设备控制」总开关，聊天里也不再有这个能力，且立即停止。
